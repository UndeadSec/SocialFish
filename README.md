### PREREQUISITES

* Python 2.7 
* Wget from Python
* PHP
* sudo

### TESTED ON
* **Kali Linux - Rolling Edition**

* **Linux Mint - 18.3 Sylvia**

* **Ubuntu - 16.04.3 LTS**

* **MacOS High Sierra**

### CLONE
```
git clone https://github.com/UndeadSec/SocialFish.git
```

### RUNNING

```
cd SocialFish
```

```
sudo pip install -r requirements.txt
```

```
python SocialFish.py
```
